3. HTML Architecture

Every HTML file features a standardized structural outline containing three major distinct semantic blocks:

  • <header class="main-header">: Houses the branding logotype and navigation system links.
  • <main>: Contains custom section-specific page bodies (e.g., banner, lists, grids).
  • <footer class="main-footer">: Includes generic legal notices, operational timing sheets, and social networks link anchors.

Sections are distinctly separated and labeled with explicit structural comments to make jumping between components easy.

4. CSS & Design Variables

All aesthetic configurations are centralized inside css/style.css. The project leverages CSS Custom Properties (Variables) defined at the :root level for seamless branding switches:

:root {
    --primary-color: #c5a880; /* Elegant Restaurant Gold Color */
    --secondary-color: #1a1a1a; /* Premium Dark Background Color */
    --text-color: #333333;
    --light-bg: #f9f6f0;
    --white: #ffffff;
    --font-heading: 'Playfair Display', serif;
    --font-body: 'Poppins', sans-serif;
}

To implement global layout revisions or swap background presets, simply update these values directly inside the variables block.

5. JavaScript & Interactive Elements

The core script layer resides inside js/main.js, built cleanly using pure vanilla Javascript (ES6) for zero dependencies. It governs:

  • Responsive Mobile Overlay Menu: Toggles visibility of the navigation sheet automatically on viewports smaller than 768px.
  • Smooth Scroll Anchoring: Provides an organic deceleration animation effect when link anchors are triggered.
  • Dynamic Header Adaptation: Computes the scrolling offset dynamically to apply background tint changes to the main navbar header.

6. How to Customize

Changing the Logo

Open any .html file and find the branding anchor line: <a href="index.html" class="logo">Gourmet<span>Haven</span></a>. Change the inner text content or drop in your standard <img> asset directly inside that block.

Modifying Menu Items or Prices

Navigate to menu.html. Each food item is wrapped inside a <div class="menu-item"> container. Simply swap the placeholder image URL, rename the text node inside the <h3> tag, and adjust the price tag values inside the <span class="price"> wrapper.
